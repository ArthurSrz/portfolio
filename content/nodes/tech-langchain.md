# LangChain

LLM orchestration framework for building AI applications with chains, agents, and retrieval-augmented generation.

## Capabilities

- **Chain Construction**: Building multi-step LLM workflows
- **RAG Integration**: Retrieval-augmented generation pipelines
- **Agent Design**: Autonomous AI agent architectures
- **Tool Use**: Enabling LLMs to use external tools

## Applications

Used extensively in interpretability prototypes:
- **Inside LLMs Mind**: Visualizing chain routing and tool selection
- **The New Sentiment**: Multi-topic sentiment analysis pipelines
- **Borges Graph**: Knowledge graph integration with LLMs
