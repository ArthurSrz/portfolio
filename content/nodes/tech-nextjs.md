# Next.js

React framework for production-grade web applications with server-side rendering and API routes.

## Capabilities

- **App Router**: Modern React Server Components
- **API Routes**: Backend logic within the same codebase
- **Edge Functions**: Serverless deployment
- **Vercel Integration**: Zero-config deployment
