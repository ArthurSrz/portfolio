# Neo4j

Graph database platform for knowledge graphs, semantic data storage, and graph-based RAG systems.

## Capabilities

- **Cypher Queries**: Native graph query language mastery
- **Data Modeling**: Property graph schema design
- **Graph Data Science**: Community detection, centrality algorithms
- **Integration**: Python drivers, GraphRAG patterns

## Applications

Used in knowledge graph projects like Borges Graph, and for building interpretable AI systems that require graph-based navigation and reasoning.
